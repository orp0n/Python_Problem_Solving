Python Problem Solving - Kickstart 1

Welcome to the "Python Problem Solving - Kickstart 1" repository! This project aims to enhance your Python skills through various problem-solving exercises.

Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/your-username/python-problem-solving-kickstart-1.git
   ```
2. Navigate to the project directory:
   ```
   cd python-problem-solving-kickstart-1
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

Problems and Solutions

Problems are categorized by difficulty: Easy, Medium, Hard. Try solving them before checking the solutions.
